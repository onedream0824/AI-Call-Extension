AI Call Assistant

Configure your backend API and keyboard shortcut for interview support.

Server configuration

Your API receives a POST with the captured caption and returns a JSON (or plain text) response.

Caption capture

Works with Google Meet, Teams, Zoom, and other pages that show live captions in the DOM. Add a custom CSS selector if needed.

Hotkey

Press the hotkey during a call to capture the latest caption, send it to your API, and show the response in the sidebar.

Ctrl+Shift+Y

Default shortcut. Change it in your browser’s extension shortcuts page.

Chrome: chrome://extensions/shortcuts · Edge: edge://extensions/shortcuts

API contract example

POST /your-endpoint
Content-Type: application/json

{ "text": "caption from live subtitles" }

// Response
{ "response": "Suggested answer or translation for the freelancer" }
