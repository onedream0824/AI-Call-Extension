AI Call Assistant

Enter your backend URL and keyboard shortcut for interview support.

Backend

Captions are sent to this URL. Auth and other settings are handled on your server.

Caption capture

Works with Google Meet, Teams, Zoom, and other pages that show live captions in the DOM. Add a custom CSS selector if needed.

Hotkey

Press the hotkey during a call to capture the latest caption, send it to your API, and show the response in the sidebar.

Ctrl+Shift+Y

Default shortcut. Change it in your browser’s extension shortcuts page.

Chrome: chrome://extensions/shortcuts · Edge: edge://extensions/shortcuts

API contract example

GET  /threads
{ "threads": [{ "id": "...", "title": "Acme interview", "updatedAt": 1710000000 }] }

POST /threads  (multipart/form-data)
  resume: file (PDF or DOCX, required)
  jobDescription: text (required)
  title: string (optional)
→ { "thread": { "id": "...", "title": "..." } }

GET  /threads/:id/history
{ "messages": [{ "id": "...", "caption": "...", "response": "...", "createdAt": 1710000000 }] }

POST /assist
{ "text": "caption from live subtitles", "threadId": "..." }
{ "response": "...", "message": { "id": "...", "caption": "...", "response": "...", "createdAt": 1710000000 } }
// message is optional; extension appends to cached history
